## 📌 Описание

Программа для захвата скриншотов с последующим выделением произвольной области (ROI) и сохранением в формате PNG. Поддерживает многомониторные конфигурации (до 4+ экранов), включая нестандартное расположение мониторов (например, с отрицательными координатами).

## ✨ Возможности

* 🖥️ Выбор экрана из списка (все экраны / конкретный монитор)
* 🖱️ Интерактивное выделение ROI мышью (с визуальной обратной связью)
* 💾 Сохранение в PNG с автоматическим именем и настраиваемой папкой
* 📋 Копирование в буфер обмена (как изображение)
* ⚙️ Конфигурация (папка сохранения сохраняется между запусками)
* ⌨️ Горячие клавиши: Ctrl+S — сохранить, Ctrl+R — обновить, Ctrl+Shift+C — копировать

## 🛠️ Требования

Компонент	Версия	Примечание
* Python	≥ 3.7	Рекомендуется 3.9+
* wxPython	≥ 4.2.1 (Phoenix)	Обязательно! Старые версии не поддерживают wx.Display
* Pillow	≥ 9.0	Для работы с изображениями

## 💡 Linux/macOS: 

Установка wxPython может потребовать системных библиотек (libgtk-3-0, libwebkit2gtk-4.0-37, libnotify4 и др.)

## 📦 Установка и запуск

### 🔧 Вариант 1: Сборка из исходников (для разработчиков)
1. Скачайте репозиторий
bash
git clone https://github.com/alexndavydov/screenshot-roi.git
cd screenshot-roi
2. Установите зависимости
bash
pip install -r requirements.txt
3. Запустите
bash
python screenshot_roi.py

### 📦 Вариант 2: Готовая сборка (для пользователей)

✅ Предкомпилированная версия (screenshot_roi.exe) доступна в разделе Releases GitHub.

**Как использовать:**</br>
Перейдите на https://github.com/AlexDavydov357/Screenshot_Roi/releases/tag/screenshot </br>
Скачайте screenshot_roi.exe из latest release.</br>
Запустите двойным кликом — не требует Python или установки зависимостей!</br>
Работает на Windows 7+ (64-bit рекомендуется).</br>
⚠️ При первом запуске Windows Defender может показать предупреждение — нажмите «Дополнительные сведения» → «Запустить всё равно» (если доверяете источнику).

### 🚀 Как использовать

Запустите программу — появится окно с полным скриншотом.</br>
Выберите экран из выпадающего списка (верхняя панель):</br>
🖥️ Все экраны (объединённый)</br>
Экран 1 — 1920x1080 @ (0, 0)</br>
Экран 2 — 2560x1440 @ (1920, -360) и т.д.</br>
**Выделить область:**</br>
Зажмите левую кнопку мыши и потяните — появится красный прямоугольник.</br>
Отпустите кнопку — прямоугольник станет зелёным (ROI сохранён).</br>
**Сохраните или скопируйте:**</br>
💾 Сохранить ROI в PNG — сохранит в папку ~/Screenshots/ROI/ (или выбранную вами)</br>
📋 Копировать в буфер обмена — вставьте в Photoshop, Word, Telegram и т.д.</br>
Обновите скриншот (если изменилось содержимое экрана) — кнопка 🔄.</br>

### 💡 Горячие клавиши:

Ctrl+S — сохранить</br>
Ctrl+R — обновить скриншот</br>
Ctrl+Shift+C — копировать в буфер обмена</br>

## 📁 Структура проекта

screenshot-roi/</br>
├── screenshot_roi.py          # Основной файл программы (исходный код)</br>
├── README.md                  # Этот файл</br>

</br>
{</br>
  "save_dir": "C:\\Users\\User\\Screenshots"</br>
}</br>


## 🧪 Разработка и отладка

Для отладки координат экранов раскомментируйте print(...) в _get_monitors() и _capture_screen().</br>
Логи выводятся в консоль при запуске (например, [DEBUG] Обнаружено мониторов: 3).</br>
Программа автоматически масштабирует изображение под окно, сохраняя aspect ratio.</br>
