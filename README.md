📌 Описание
Программа для захвата скриншотов с последующим выделением произвольной области (ROI) и сохранением в формате PNG. Поддерживает многомониторные конфигурации (до 4+ экранов), включая нестандартное расположение мониторов (например, с отрицательными координатами).

✨ Возможности
🖥️ Выбор экрана из списка (все экраны / конкретный монитор)
🖱️ Интерактивное выделение ROI мышью (с визуальной обратной связью)
💾 Сохранение в PNG с автоматическим именем и настраиваемой папкой
📋 Копирование в буфер обмена (как изображение)
⚙️ Конфигурация (папка сохранения сохраняется между запусками)
⌨️ Горячие клавиши: Ctrl+S — сохранить, Ctrl+R — обновить, Ctrl+Shift+C — копировать
🛠️ Требования
Компонент	Версия	Примечание
Python	≥ 3.7	Рекомендуется 3.9+
wxPython	≥ 4.2.1 (Phoenix)	Обязательно! Старые версии не поддерживают wx.Display
Pillow	≥ 9.0	Для работы с изображениями

💡 Linux/macOS: Установка wxPython может потребовать системных библиотек (libgtk-3-0, libwebkit2gtk-4.0-37, libnotify4 и др.)

📦 Установка и запуск
🔧 Вариант 1: Сборка из исходников (для разработчиков)
1. Скачайте репозиторий
bash
git clone https://github.com/alexndavydov/screenshot-roi.git
cd screenshot-roi
2. Установите зависимости
bash
pip install -r requirements.txt
3. Запустите
bash
python screenshot_roi.py
📦 Вариант 2: Готовая сборка (для пользователей)
✅ Предкомпилированная версия (screenshot_roi.exe) доступна в разделе Releases GitHub.

Как использовать:
Скачайте screenshot_roi.exe из latest release.
Запустите двойным кликом — не требует Python или установки зависимостей!
Работает на Windows 7+ (64-bit рекомендуется).
⚠️ При первом запуске Windows Defender может показать предупреждение — нажмите «Дополнительные сведения» → «Запустить всё равно» (если доверяете источнику).

🚀 Как использовать
Запустите программу — появится окно с полным скриншотом.
Выберите экран из выпадающего списка (верхняя панель):
🖥️ Все экраны (объединённый)
Экран 1 — 1920x1080 @ (0, 0)
Экран 2 — 2560x1440 @ (1920, -360) и т.д.
Выделите область:
Зажмите левую кнопку мыши и потяните — появится красный прямоугольник.
Отпустите кнопку — прямоугольник станет зелёным (ROI сохранён).
Сохраните или скопируйте:
💾 Сохранить ROI в PNG — сохранит в папку ~/Screenshots/ROI/ (или выбранную вами)
📋 Копировать в буфер обмена — вставьте в Photoshop, Word, Telegram и т.д.
Обновите скриншот (если изменилось содержимое экрана) — кнопка 🔄.
💡 Горячие клавиши:

Ctrl+S — сохранить
Ctrl+R — обновить скриншот
Ctrl+Shift+C — копировать в буфер обмена
📁 Структура проекта
screenshot-roi/
├── screenshot_roi.py          # Основной файл программы (исходный код)
├── README.md                  # Этот файл
├── requirements.txt           # Зависимости
├── screenshot_roi.exe         # Готовая сборка (Windows, x64)
└── .screenshot_roi_config.json  # Конфиг (создаётся автоматически в домашней папке)
Конфиг (%USERPROFILE%\.screenshot_roi_config.json в Windows) хранит:
{
  "save_dir": "C:\\Users\\User\\Screenshots"
}

🧪 Разработка и отладка
Для отладки координат экранов раскомментируйте print(...) в _get_monitors() и _capture_screen().
Логи выводятся в консоль при запуске (например, [DEBUG] Обнаружено мониторов: 3).
Программа автоматически масштабирует изображение под окно, сохраняя aspect ratio.
